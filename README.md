# Spaceship configurator

Demo project from Figma design


## Usage


```bash
# Clone repository
$ git clone https://github.com/Lamanche/spaceship-configurator
```
NPM:

```bash
# Install dependencies
$ npm install

# Start development server
$ npm start
```

Yarn:

```bash
# Install dependencies
$ yarn install

# Start development server
$ yarn start
```


## Technologies used

-   React.js
-   Sass
